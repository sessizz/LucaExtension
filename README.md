# Luca Captcha ve Firma Doldurucu

Chrome uzantisi; `https://agiris.luca.com.tr/` adresindeki Luca giris ekraninda captcha cozumu ve kayitli firma bilgilerini otomatik doldurma icin kullanilir.

## Ozellikler

- Luca giris sayfasinda firma secim paneli gosterir.
- Uye numarasi, kullanici adi ve parola alanlarini kayitli firma bilgileriyle doldurur.
- Doldurma sonrasinda giris butonuna tiklar.
- Captcha cozumu icin kullanicinin kendi OpenRouter API anahtarini kullanir.
- Ayarlari ve firma bilgilerini Chrome `storage.local` alaninda saklar.

## Kurulum

1. Chrome'da `chrome://extensions` sayfasini acin.
2. Gelistirici modunu etkinlestirin.
3. `Paketlenmemis oge yukle` ile bu klasoru secin.
